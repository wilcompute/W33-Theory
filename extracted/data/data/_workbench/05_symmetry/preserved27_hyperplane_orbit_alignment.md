Preserved-27 automorphism orbits (D4):

Hyperplane-27 D8 orbits:
- sizes: 8,4,4,4,4,2,1 (see `hyperplane27_orbits_under_D8.md`)

This size-level match shows the preserved-27 subgraph carries the same D8 orbit structure as the hyperplane 27.

this is now a well-posed (finite) group-conjugacy problem.
